WeatherApp

WeatherApp is an Angular + Spring application remade from Hibernate version of it.

App serve for weather forecast checking in any location given by city name or latitude and longitude coordinates.

It gives access to 7 days forward forecast and searching history for logged in users

Weather APIs

  • Application uses weather apis listed below:
    • Weatherbit API
    • OpenWeather API

Input data

Standard functionality lets user check tomorrows forecast for given location by:

  • name of city,
  • latitude and longitude coordinates values.

Signed in users are capable of checking forecast for 7 days forward by picking a date with datepicker.

Output data

App responses with parameters below:

  • temperature (Celsius degree),
  • pressure (hPa),
  • humidity (%),
  • wind speed (m/s),
  • wind degree (degree).

If app cannot find given location, it outputs the message with error.
